Tonight's Git(hub) Tutorial

  • Manage a project with Git
  • Collaborate on a Git managed project
  • Set up a repo on Github
  • Get involved in open source Github projects

Firstly, you have to download Git

http://git-scm.com/downloads

Then, set it up!

  • git config --global user.name "FirstName LastName"
  • git config --global user.email "myname@email.com"
  • git config --global color.ui true
  • git config --list

Now, some basic Git commands

  • git init
  • git diff
  • git add
  • git reset HEAD myfilename
  • git commit -m
  • git status
  • git checkout ...
  • git log -5

Collaborating Commands

  • git push
  • git fetch
  • git pull (== fetch + merge)
  • git branch
  • git merge

Setting up your Repo

  • mkdir Cool-Project
  • touch README
  • add, commit
  • Create repo on your Github account
  • git remote add origin https://github.com/MzTek-/Cool-Project
  • git push origin master
  • .git/config

Some Good Git Guidelines

  • Always fetch before you work (find out what’s new);
  • Fetch before you push;
  • Fetch often (it’s not destructive, so do it often)
  • Write good commit messages: Describe the changes you make in a commit set — label it clearly, descriptively; write in the present - not past - tense; break commits into small, cohesive changes

To merge your master and origin/master branches:
git diff origin/master master (to see what’s different)
git merge origin/master

Contributing to Open Source projects on Github

  • Check the issues log and post an issue
  • Click “Fork” and clone the project locally
  • Check what the commit message conventions are on the project
  • Push it back up to the repo, click “Pull Request” on the github page.
  • If the Github admin likes the changes, they will merge your changes into the main repo

Key Points to Remember

  • Merge conflicts can be a b*?$!
  • Write elegant commit messages
  • Use a .gitignore file to ignore Mac dust like .DS_store files, compiled source code, packages, compressed files, etc.
  • Git is not as useful for tracking non-text files or code that requires software
